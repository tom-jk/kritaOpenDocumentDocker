Open Documents Docker

A Krita docker that provides a compact list of your open images and can switch between them.

Usage

Left-click on the thumbnail of an image to switch to it.
Subsequent clicks cycle through its views in the current window.

Right-click for a menu of common file actions: save, save as, export, etc.
You can also manage the documents views.

Settings can be adjusted by clicking the 'hamburger' menu button.
Most settings have tooltips that describe what they do.

You can set how and when thumbnails are updated.
Refresh on save setting: update the thumbnail when the document is saved.
Refresh periodically setting: update after the document has changed.
Refresh manually: you can use the button in the docker to update on demand.

Context Menu

Besides regular krita actions, ODD provides two additional actions:

Revert: close the document and reload it from disk.
You have the option to continue using the existing views.

Quick Copy Merged: Quickly copy whole image to clipboard.
switches to the document and performs the following actions:
select all, copy merged, undo (copy merged), undo (select all).
